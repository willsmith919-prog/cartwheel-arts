# Cartwheel Arts

Kids art class registration and studio site for [cartwheelarts.org](https://cartwheelarts.org). Stack: React, Vite, Tailwind, Firebase, Stripe (planned), Vercel.

- **Branding:** edit `src/theme.js` for colors and font stacks.
- **Deploy & services:** follow [docs/SETUP.md](docs/SETUP.md) for GitHub, Firebase, Vercel, Stripe, and Resend.

```bash
npm install
cp .env.example .env.local   # add Firebase web config
npm run dev
```
